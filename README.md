
# Hanzi Shifu Restaurant

## Description
Hanzi Shifu Restaurant is a ReactJS web application designed to showcase a restaurant’s services and allow users to make reservations.  
The app is fully responsive and works on both desktop and mobile. It includes five main pages plus login/signup functionality integrated through the reservation page.

---

## Pages Overview
- **Home:** Landing page with restaurant overview and highlights  
- **Menu:** Displays the restaurant’s menu items  
- **Location:** Shows the restaurant location and contact information  
- **Gallery:** Image gallery showcasing the restaurant and dishes  
- **Reservation:** Allows users to book a table. Users must **login or signup** before making a reservation  
- **Login / Signup:** Accessed through the Reservation page; required for users to reserve a table

---

## User Flow
1. User navigates to the **Reservation** page  
2. If not logged in, the **Login / Signup modal/page** appears  
3. After logging in, the user can fill out the reservation form  
4. Reservation is confirmed and optionally stored or displayed on the dynamic page

---

## Technologies Used
- ReactJS  
- Bootstrap 
- React Router (for navigation)  
- Git (version control)
